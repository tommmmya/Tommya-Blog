    浏览器输入url会发生什么

    主要包括以下几个基本步骤：

    1. 浏览器的地址栏输入 URL 并按下回车
    2. 浏览器查找当前 URL 是否存在缓存，并比较缓存是否过期
    3. DNS 解析 URL 对应的 IP
    4. 根据 IP 建立 TCP 连接（三次握手）
    5. 发送 HTTP 请求
    6. 服务器处理请求
    7. 返回 HTTP 响应
    8. 浏览器接受 HTTP 响应
    9. 关闭 TCP 连接（四次挥手）
    10. 浏览器解析 HTML
    11. 浏览器布局渲染
